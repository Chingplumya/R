# 一、项目简介
本项目基于 R 语言survival包内置 lung 数据集，完成一套完整的晚期肺癌基线指标生存预后分析，全流程包含营养指标缺失值预处理、Kaplan-Meier 分层生存分析、单 / 多因素 Cox 比例风险回归、独立预后因素森林图可视化，所有代码可一键复现。
## 完整分析流程
### 1.数据预处理与缺失插补
针对体重丢失、日均热量摄入两项缺失率不同的营养指标，采用有序级联残差抽样插补：先利用临床基线变量填补低缺失的体重丢失指标，再将填补后的体重数据作为协变量填充热量指标；同时增加生理数值边界约束，保证插补结果符合临床逻辑，全程保留全部样本，不做数据删除。
### 2.单因素 Cox 批量筛选
对全部候选变量批量构建单变量 Cox 模型，统一提取风险比 HR、95% 置信区间、Wald 检验 P 值，快速筛选潜在预后因子。
### 3.Kaplan-Meier 生存分层曲线
分别按性别、合并 ECOG 体能状态绘制分层 KM 曲线，通过 Log-rank 检验对比组间生存差异。
### 4.多因素 Cox 回归建模
遵循简约建模原则，剔除存在多重共线性的同源体能指标、效应微弱且仅临界显著的年龄变量，最终构建「性别 + ECOG 分层」精简双变量模型，校正混杂作用，筛选独立预后因素。
### 5.森林图可视化
将多因素回归风险效应可视化，直观对比各组死亡风险大小，与 KM 分层曲线结论交叉印证。
## 核心研究结论
1.女性性别是晚期肺癌总生存的独立保护因素；

2.ECOG 体能状态存在梯度风险效应：患者活动能力越差，长期死亡风险越高；

3.体重丢失、日均热量两项营养指标无独立预后预测价值。

# 四、关键建模方案说明
## 1.精简双变量 Cox 模型优势
规避多重共线性：自评 KPS 与医生评估 ECOG 均反映患者体能，同时纳入会造成系数估计不稳定；

剔除弱效应变量：年龄仅临界显著，风险比幅度极小，无实际临床预后意义；

聚焦肿瘤领域经典核心预后标志物（性别、ECOG），临床解读简洁，与两张核心 KM 分层分析完美对应。
## 2.缺失插补策略优势
未采用简单删除、均值填充，级联插补保留样本总量，同时维持营养指标与临床协变量的内在关联，数据真实性更强。

# 五、分析局限性
未针对营养指标插补策略开展多重插补敏感性分析；

数据集缺少病理分期、治疗方案等高影响力协变量，模型区分度中等（C-index=0.644）；

未补充 Cox 模型比例风险（PH）假设检验；

未构建四变量全模型作为对照，无法对比两种模型结论差异。

# 六、后续优化拓展方向

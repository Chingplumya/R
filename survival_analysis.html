1 数据准备

使用survival包内置lung肺癌随访数据，共228例晚期肺癌患者，探究性别、年龄、体能评分等对生存预后的影响。

1.1 数据集变量介绍

inst，机构代码 (Institution code) 表示患者所在的治疗中心编号。通常作为随机效应或控制变量，但在本小样本分析中常被忽略。

time，生存时间 (Survival time) 目标变量之一。记录从入组到死亡或最后一次随访的天数。单位：天 (Days)

status，生存状态 (Censoring status) 目标变量之二。指示观察结束时患者的状态。 1 = 删失 (Censored): 患者仍存活或失访 2 = 死亡 (Dead): 事件发生

age，年龄 (Age in years)。单位：岁 (Years)

sex，性别。1 = 男 (Male)，2 = 女 (Female)

ph.ecog，ECOG 体力评分 (ECOG performance score) 关键预测因子。评估患者日常活动能力，分数越高身体状况越差。 0: 正常活动 1: 轻微症状，能从事轻体力活动 2: 白天卧床<50% 3: 白天卧床>50% 4: 完全卧床 5: 死亡

ph.karno，医生评分的 Karnofsky 评分 (Karnofsky score rated by physician)，评估患者功能状态的百分制评分。 分数越高越好（与 ECOG 相反）。范围：0-100 (100=完全健康)

pat.karno，患者自评的 Karnofsky 评分 (Karnofsky score rated by patient)，患者对自己身体状况的主观评分。 常与医生评分对比，分析主观感知对预后的影响。范围：0-100

eal.cal，每日卡路里摄入量 (Calories consumed at meals) 营养状况指标。营养不良是癌症患者预后不良的重要标志。单位：千卡/天 (kcal/day)

wt.loss，体重丢失 (Weight loss in last 6 months) 过去 6 个月内体重下降的磅数。单位：磅 (lbs)。 负值 = 增重，正值 = 减重

缺失值填补策略对比与决策总结

结果显示，ph.ecog与ph.karno高度负相关（|r|>0.7），信息重叠严重，这里剔除ph.karno。

针对营养指标 wt.loss、meal.cal 线性相关性弱（|r|<0.2）、缺失程度分层的特征，对比 MICE 多重插补与有序级联残差抽样回归插补的适配性。

决策依据：本数据集基线协变量数量有限，营养指标预测信噪比偏低；标准 MICE 依赖多轮迭代预测，在弱预测信息下无法提升填补精度，还会放大原生数据噪声；本研究采用模型真实残差有放回抽样作为扰动项，不强制正态分布假设，能完整复刻原始数据残差分布，兼顾分布保真与轻量化计算。

实施流程：有序级联填补，先处理低缺失率 wt.loss（缺失 6%），通过线性模型预测缺失样本期望值，抽取模型原始残差叠加至预测值完成填充；再将填补完成的 wt.loss 作为新增协变量，重复残差抽样逻辑填补高缺失率 meal.cal（缺失 20%）。全程保留全部 208 例有效样本，生成完整无缺失数据集。

wt.loss预测线性回归模型结果解读与填补方案支撑

本模型以完整样本的年龄、性别、ECOG评分、患者自评KPS为自变量，构建多元线性回归预测体重丢失wt.loss，为后续缺失值插补提供预测基准。

  1. 整体有效性：模型全局F检验P<0.05，自变量联合具备显著预测作用，可作为缺失值预测基准；但校正决定系数仅0.0385，现有基线变量仅能解释3.85%的体重丢失变异，模型预测效能偏弱。
  2. 残差分布特征：模型残差区间跨度极大，存在明显偏态与极端离群值，不符合标准正态分布；若仅用模型预测值直接填充缺失，会大幅压缩数据离散度、丢失极端减重/增重样本。
  3. 变量预测贡献：仅性别、ECOG体能评分呈现边缘显著关联，年龄、患者自评KPS无预测价值，缺少营养、治疗等强关联协变量进一步限制模型精度。
  4. 方案选择依据：基于上述模型短板，本研究放弃正态随机噪声扰动，采用有放回抽样提取模型原始残差叠加至预测值的插补策略；该方法不强制正态假设，完整复刻原始残差的偏态分布与极端波动，最大程度还原真实wt.loss数值分布，兼顾样本完整性与数据分布保真度。

meal.cal热量摄入预测模型与缺失填补结果解读

本模型为级联填补第二步，以年龄、性别、ECOG医生评分、患者自评KPS、已完成填补的体重丢失wt.loss作为自变量，构建多元线性回归预测日均卡路里摄入meal.cal，用于20%中度缺失样本的插补。

  1. 模型整体性能：全局F检验P远小于0.001，模型高度显著；校正决定系数0.1133，相比仅使用基线临床变量的wt.loss模型预测能力大幅提升，验证了“先填补体重指标、再纳入热量预测模型”的有序级联策略可有效提升填补精度。
  2. 变量预测贡献：年龄、性别为极强独立预测因子，患者自评KPS存在弱正向关联；ECOG评分、体重丢失无单独显著关联，但wt.loss纳入后改善模型整体拟合效果。
  3. 残差分布特征：模型残差呈明显右偏，存在大量极端正向残差，不满足正态假设；因此采用抽取模型原生残差叠加预测值的扰动方案，相比人为正态噪声更贴合原始数据离散特征。
  4. 生理约束处理：预测叠加残差后可能生成无生理意义的负热量值，通过pmax函数设置下界0截断，剔除不合理填补数值，保证所有观测符合临床常识。
  5. 填补后分布校验：完成插补后meal.cal值域96~2600千卡，四分位数分布与原始完整样本特征一致，既保留极端高低摄入样本，又不存在负值失真，填补分布保真效果良好。

研究局限性

① 残差抽样采用有放回抽取，缺失样本量大时可能造成分布轻微收缩； ② 暂未开展插补方案敏感性分析，缺少 MICE 多重插补对照，无法定量评估不同填补策略对后续生存模型系数、P 值的扰动。

  4. 风险表样本损耗：基线男性样本137例、女性90例；各随访节点男性病例损耗比例更高，随访至1000天时女性全部发生死亡终点，仍存在2例男性截尾观测。
  5. 分析小结：性别是显著的单变量预后因素，女性晚期肺癌患者整体生存预后优于男性，后续将通过多因素Cox回归校正年龄、体能评分等混杂变量，验证性别是否为独立保护因素。

2.2 单因素Cox回归定量分析

循环运行单因素模型，然后逐个进行分析。 注意到二分类及连续变量，无需特殊处理； 多分类变量（有多行系数），策略是提取风险最高的一组（通常是最后一组，即Level 2-3 vs 0）作为代表

candidates <- c("sex", "ecog_group", "age", "pat.karno", "meal.cal", "wt.loss")
univariate_results <- lapply(candidates, function(var) {
  formula <- as.formula(paste("surv_obj ~", var))
  cox_model <- coxph(formula, data = df_final)
  summary_cox <- summary(cox_model)
  
  p_val_global <- summary_cox$waldtest["pvalue"]
  
  coef_tab <- summary_cox$coefficients
  conf_tab <- summary_cox$conf.int
  n_rows <- nrow(coef_tab)
  
  if (n_rows == 1) {
    # 情况 A: 二分类 或 连续变量 (只有 1 行系数)
    hr_val <- exp(coef_tab[1, 1])
    lower_val <- conf_tab[1, 3]
    upper_val <- conf_tab[1, 4]
    display_hr <- round(hr_val, 3)
    display_ci <- paste0(round(lower_val, 3), "-", round(upper_val, 3))
  } else {
    # 情况 B: 多分类变量 (有多行系数)
    last_row <- n_rows
    
    hr_val <- exp(coef_tab[last_row, 1])
    lower_val <- conf_tab[last_row, 3]
    upper_val <- conf_tab[last_row, 4]
    
    display_hr <- round(hr_val, 3)
    display_ci <- paste0(round(lower_val, 3), "-", round(upper_val, 4))
  }
  
  # 提取关键指标
  data.frame(
    Variable = var,
    HR = display_hr,
    CI_95 = display_ci,
    P_Value = round(p_val_global, 4),
    Significant = ifelse(p_val_global < 0.05, "Yes", "No")
  )
})
# 合并结果并排序
results_df <- bind_rows(univariate_results) %>%
  arrange(P_Value)

双变量精简Cox模型 vs 四变量全变量模型对比分析

单因素筛选共得到4项显著变量（sex、ecog_group、age、pat.karno），本研究优先构建性别+ECOG分层双变量精简模型，建模决策优势如下：

  1. 规避体能指标多重共线性：pat.karno（自评KPS）与ecog_group（医生ECOG）同源反映患者活动能力，同时纳入4变量模型会出现共线性，压缩变量效应、放大标准误；精简模型仅保留分层梯度更清晰、效应更强的ECOG分组，消除共线性偏差。
  2. 遵循简约建模原则：age仅临界显著，HR=1.019效应幅度微弱，无实际临床预后价值；pat.karno为次要体能指标，二者仅能小幅提升模型C-index，增益有限；剔除弱效应冗余变量后模型更轻量化。
  3. 临床解读逻辑清晰：性别、ECOG体能状态是晚期肺癌公认核心预后标志物，双变量模型聚焦核心风险通路，叙事简洁直观，避免次要变量干扰结论解读。
  4. 分析链条完整闭环：前期分别完成性别、ECOG分层KM单因素生存分析，双变量模型恰好对应两项分层指标，实现「单分层观察差异—多因素校正混杂验证独立效应」的完整分析流程，报告逻辑连贯统一。

模型局限性：未纳入age、pat.karno作为混杂校正变量，无法评估两项次要指标的独立预后作用，模型区分度C-index存在小幅损耗；若需全面校正全部基线混杂，可拓展构建四变量完整模型作为敏感性对照。

分析局限

  1. 未检验Cox模型比例风险（PH）假设，无法验证各变量风险比是否随随访时间恒定；
  2. 营养指标wt.loss、meal.cal单因素无统计学意义未纳入模型，缺失值采用单轮级联残差插补，无敏感性分析验证填补策略对回归系数的扰动；
  3. 缺少肿瘤分期、治疗方案、消化道症状等强预后协变量，模型C指数区分能力存在上限。

4 总结

本项目完成晚期肺癌数据集完整数据分析，流程分为数据预处理与生存预后建模两大模块。

预处理阶段针对 wt.loss、meal.cal 两项营养指标的不同缺失比例，采用有序级联残差抽样插补策略：先依托基线临床变量预测低缺失率体重丢失指标，再将填补后的体重数据纳入模型完成中度缺失热量指标填充，同时增加生理数值边界约束保证填补结果符合临床逻辑，全程完整保留全部有效样本，生成无缺失分析数据集。

预后分析部分，先通过 Kaplan-Meier 分层生存曲线直观观察性别、ECOG 体能状态带来的生存分层差异；批量单因素 Cox 回归初筛预后变量后，结合多重共线性与临床效应幅度精简模型，剔除同源体能指标与效应微弱的年龄变量，构建性别 + ECOG 分层多因素 Cox 回归；校正混杂后证实女性为独立生存保护因素，患者体能越差死亡风险呈梯度升高，配套森林图可视化各组风险效应并与生存曲线结论交叉印证。分析结果显示体重丢失、日均热量摄入无预后预测价值，仅性别与 ECOG 体能状态可稳定预判晚期肺癌患者长期生存结局。
